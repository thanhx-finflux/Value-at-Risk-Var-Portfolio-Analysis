# Value-at-Risk-Var-Portfolio-Analysis
This project is comprehensive Python-base Value at Risk VaR) calculator for stock portfolios using historical data from Yahoo Finance. The project demonstrates financial risk analysis, portfolio optimization, and data visualization techniques.

### Project Overview
The analysis is performed on a diversified portfolio of 8 stocks including Apple (AAPL), Microsoft (MSFT), Alphabet (GOOGL), Amazon (AMZN), Tesla (TSLA), Johnson & Johnson (JNJ), Visa (V), and NVIDIA (NVDA).

VaR is a statistical measure used to assess the level of financial risk within a portfolio over a special time frame.
This project calculates VaR using historical simulation methods and provides risk analytics including:
- Historical VaR: calculations across multiple time horizon (daily, monthly, annual)
- Risk metric summary
- Real-time stock data from Yahoo Finance
- Portfolio performance visualization
